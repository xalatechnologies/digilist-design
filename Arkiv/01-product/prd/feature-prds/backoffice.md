## Overview

The Backoffice application provides administrative functionality for case handlers and tenant administrators.

---

## Key Features

- Booking approval workflows
- Listing management
- User management
- Tenant configuration
- Reporting and analytics

---

## User Roles

- **Case Handler** - Processes booking requests
- **Tenant Admin** - Manages tenant configuration
- **Listing Manager** - Manages listings and zones
